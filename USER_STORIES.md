___
##User Stories
___
```
As a User
So that I can see my saved bookmarks
I want to see a list of links
```
```
As a User
So that I can save a new bookmark
I want to add my new bookmark to the list of links
```
```
As a User
So that I can find similar bookmarks later
I want add tags when I add my new bookmark to the list of links
```
```
As a User
So that I can find bookmarks with the same tag
I want to filter my bookmarks by their tags
```
